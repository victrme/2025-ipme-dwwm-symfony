## 3. Et on continue d'aller plus loin ?


Simuler la logique d'achat d'un jeu, pour simplifier les comportement un utilisateur ne peut acheter un jeu qu'une seule fois ! 
Il faut implémenter la logique suivante :


### 3.1 Action dans "show_game" (côté front)

- Implémenter une fonction Twig de nom "doesUserOwnGame" :
  - Elle doit vérifier si un jeu est présent dans la bibliothèque de l'utilisateur connecté
  - Elle voit un booléan true/false
- Si le jeu est dans la bibliothèque de l'utilisateur, alors on affiche un icone en mode "Vous possédez ce jeu"
- Sinon, on met un bouton avec un icône de chariot (cart ?) pour ajouter le jeu dans le panier de l'utilisateur
(Le bouton d'ajout au panier ne fait rien)


### 3.2 Ajouter le comportement du bouton "ajouter au panier" 


- Il faut forcément du Typescript pour cette étape :
    - Définir une action au clic sur le bouton d'ajout au panier, qui envoie une requête AJAX en "POST" vers un nouveau "AjaxCartController" avec la méthode "addToCart"
    - Le JSON envoyé aura la clé "gameId"
    - La méthode "addToCart" ajoute le jeu en session (session de nom "CART_GAMES"), s'il n'existe pas déjà dans la session
      - A voir si vous vous l'informer ou non de si le jeu existe déjà...
- Les jeux doivent avoir au minimum les attributs suivants :
  - id
  - name
  - price
  - (image ?)


### 3.3 Faire le contrôleur "CartController"


- Il a une méthode `index` qui utilise le server `SessionCartService`.


#### 3.3.1 Avec Twig classique


- Retourner les objets de la session "CART_GAMES" dans la vue "cart/index.html.twig"
- A vous de voir si vous voulez utiliser un tableau PHP ou des pseudos objets PHP pour représenter les jeux
- Afficher les jeux dans un tableau HTML


#### 3.3.2 Avec Vue.js


- Charger les données dans un conteneur HTML d'id "vue-root-container"
- `serialializedData` représente les données JSON des jeux du panier

```html

    <div id="vue-root-container" data-cart-games="{{ serialializedData }}">

    </div>

```

Exemple de données JSON :

```json
[
    {
        "id": 1,
        "name": "Overwatch",
        "price": 24
    },
    {
        "id": 2,
        "name": "Among Us",
        "price": 3
    }
]
```
